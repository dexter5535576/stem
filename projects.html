STEM Club Webdesign Competition Spring 2018 winner
- Ying Lung (currently transfer to UC Berkeley majoring in Computer Science)


Link to his webpage



#1.  Judges will be selected from CCSF CS, Math, and Engineering Faculties.  They will each score your project based on the creativity, functions, and entiretity of the UI.  We will have a look on the code to check if there is plarigism.  (Using other people's CSS stylesheets online is not considered to be plarigism in this case)  We will not score on your codes.  The whole process will be done anonymously in order to be fair.  Your project will only be judged on PC, so don't worry about making the page to be mobile friendly. #2.  This is the first semester we hold the competition at CCSF.  We would like to see how far will CCSF students go when writing codes to design the web page.  That is one reason that we limit the competition to HTML and CSS (HTML5 and CSS5 is fine).  The second reason is we find that Javascript files can not be sent by emails.   Try to master your web design skill only using HTML and CSS.  You will learn a lot. #3.  If the code doesn't work on our computer, we will email you back.  You will have seven days to work on it again.  The project must be ok to run on every PC.  (or any laptop)   Result of the competition will come out on 5/24 on STEM webpage.  Award will be sent to the winner.
